# Completed Pieces Archive

When a piece ships, move a copy of its key artefacts here:

- The final chart HTML
- The final captions
- The source data CSV
- A note on how it performed (link to the relevant retro if useful)

Subfolder per piece, named `NNN-piece-slug/` (e.g. `001-extradition-atlas/`).

This is the permanent record. When you want to remember "how did we handle X last time?", this is where you look.
